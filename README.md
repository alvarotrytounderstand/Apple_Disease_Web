# 🍎 Apple Leaf Disease Detector

Aplikasi web berbasis AI untuk mendeteksi penyakit pada daun apel menggunakan algoritma Deep Learning (**MobileNetV2**). 

## 📊 Performa Model
Model ini dilatih menggunakan metode Transfer Learning dan mencapai akurasi **98%** dalam mengklasifikasikan 4 kondisi daun:
* Apple Scab
* Black Rot
* Cedar Apple Rust
* Healthy (Sehat)

## 🛠️ Teknologi yang Digunakan
* Python
* TensorFlow / Keras (MobileNetV2)
* Streamlit (Web Interface)
